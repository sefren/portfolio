## Features

---

## Setup

Install dependencies:
```bash
npm install
````

Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Build for production:

```bash
npm run build
npm run start
```

---

## Deployment

Deploy on **Vercel**:

```bash
vercel --prod
```
